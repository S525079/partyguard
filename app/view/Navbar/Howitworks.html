How It Works

Host User

  • -> Is the Fraternity head
  • -> Manages the Members of the House
  • -> Assigns Guards to particular party
  • -> Can change the location of House
  • -> Manages the Events

Guard User

  • -> Is the Fraternity Member
  • -> Stays sober for a particular party
  • -> Generally Guard users are Males

Basic User

  • -> Is the Fraternity Member
  • -> Women who attends a party
  • -> Can view their profile
